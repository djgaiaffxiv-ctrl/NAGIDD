# NAGIDD — NAGI OG DOWNLOADER

Descargador de vídeo y MP3 de cualquier web (YouTube, Vimeo, X/Twitter y +1800 sitios), de escritorio, local y sin límites.

- **MP3** (máxima calidad) y **MP4** de cualquier URL.
- **Playlists** a MP3, numeradas y en su carpeta (con nombre de álbum opcional).
- **Lista de vídeos**: pega muchos enlaces y se descargan todos.
- **Galería → vídeos**: pega la URL de una página y baja todos sus vídeos.
- Acelerador **aria2c** (16 conexiones), rescate para sitios tipo KVS, cookies del navegador para contenido privado, y registro de fallos.

Motor: [yt-dlp](https://github.com/yt-dlp/yt-dlp) + ffmpeg + aria2c. App Electron.
